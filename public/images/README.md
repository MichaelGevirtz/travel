## Image Guidelines

1. **Format**: WebP preferred, JPEG as fallback
2. **Quality**: 80-85% compression
3. **File Size**: Under 200KB for destinations, under 300KB for heroes
4. **Alt Text**: Always provide descriptive alt text in components
